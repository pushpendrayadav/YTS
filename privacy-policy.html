Privacy Policy

YT Summarizer – AI Notes · Last updated: March 16, 2026

1. Overview

YT Summarizer is a browser extension that generates structured Markdown summaries from YouTube video transcripts using AI. This privacy policy explains what data the extension accesses, how it is used, and how it is stored.

2. Data Collected

Data Purpose Storage
YouTube video transcript text Sent to the user's chosen AI provider to generate a summary Processed in-memory; not persisted beyond the local cache
Video metadata (title, channel, URL) Included in the generated summary file Cached locally for up to 30 days
AI provider API keys (Groq, Claude, OpenAI) Authenticates requests to AI providers on the user's behalf Stored in Chrome's encrypted sync storage
Generated summaries Cached locally to avoid regenerating for previously visited videos Stored in Chrome's local storage for 30 days, then auto-deleted

3. Data NOT Collected

  • No personal information (name, email, account details) is collected
  • No browsing history or activity outside of YouTube is accessed
  • No analytics, telemetry, or tracking of any kind
  • No cookies or fingerprinting
  • No data is sold, shared with, or transferred to third parties for advertising or any unrelated purpose

4. Third-Party Services

When the user generates a summary, the video transcript is sent to one of the following AI providers based on the user's explicit choice:

  • Groq (groq.com) — cloud API
  • Anthropic / Claude (anthropic.com) — cloud API
  • OpenAI (openai.com) — cloud API
  • Ollama (localhost only) — runs entirely on the user's machine; no data is sent externally

Transcript data is sent only to the provider the user selects, only when the user clicks "Generate summary.md", and only for the purpose of generating the summary. Each provider's own privacy policy governs how they handle data received through their API. The extension does not send data to any other remote server.

5. Local / Ollama Mode

When using Ollama, all processing happens locally on the user's device. No transcript data, API keys, or any other information leaves the machine.

6. Storage & Retention

  • API keys remain in Chrome sync storage until the user deletes them via the Settings page.
  • Cached summaries are stored locally for 30 days and are automatically cleaned up after expiry. Users can manually clear the cache at any time from the Settings page.
  • No data is stored on any external server controlled by the extension developer.

7. Permissions Justification

Permission Reason
activeTab / scripting Access the current YouTube tab to extract the video transcript
storage Store API keys and cached summaries locally in the browser
downloads Save the generated .md summary file to the user's device
Host permissions (youtube.com) Read transcript data from the YouTube page
Host permissions (API endpoints) Send transcript to the user's chosen AI provider

Any changes to this privacy policy will be posted on this page with an updated date. Continued use of the extension after changes constitutes acceptance.
